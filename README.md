# Simulador Funcional do Processador UFLA-RISC
Este projeto implementa um simulador funcional para o processador RISC de 32 bits UFLA-RISC, permitindo a simulação de um conjunto de instruções básicas, incluindo operações aritméticas, lógicas, de controle de fluxo, memória e manipulação de bits.

## Como Executar
1. Clone o repositório:
git clone https://github.com/Gustavo-Martins610/ufla-risc-simulador-grupo7

2. Compile o simulador:
cd src/simulador

No arquivo main.py, escolha mudando a variável caminho, qual arquivo quer ler
pasta binários contém testes em arquivos binários e a pasta exemplos testes em arquivos texto
rode no terminal Python main.py
